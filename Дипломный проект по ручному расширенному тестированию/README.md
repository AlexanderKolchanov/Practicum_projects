# Дипломный проект по ручному расширенному тестированию                                                                                                         ## Описание проекта
Тестирование веб‑ и мобильного приложений: проверка функциональности, валидации полей и соответствия макетам Figma.

## Цель
- Проверить реализацию требований к экранам «Статус заказа» и «Сделать заказ» в веб‑приложении.  
- Протестировать ключевые функции мобильного приложения (выделенные жирным в требованиях).  
- Выявить и задокументировать дефекты.

## Что было сделано

**Веб‑приложение:**  
- Составлен чек‑лист для экрана «Статус заказа» (вкладка «Задание 1: чек‑лист»).  
- Разработана таблица валидации полей для экрана «Сделать заказ» (вкладка «Задание 1: данные валидации»).  
- Проведено комплексное тестирование функциональности (по чек‑листам, таблицам и макетам Figma).  
- Зафиксированы баги вне тестовой документации (вкладка «Задание 1: баги вне тестовой документации»).  
- Для FAILED‑тестов созданы баг‑репорты с ID (вкладка «Баг‑репорты»).  


**Мобильное приложение:**  
- Спроектированы тест‑кейсы для функциональности, выделенной жирным в требованиях.  
- Проверена вёрстка по макетам Figma (результаты в вкладке «Задание 2: тест‑кейсы»).


## Объект тестирования
- **Веб‑приложение:** экраны «Статус заказа», «Сделать заказ» (без лендинга).  
- **Мобильное приложение:** ключевые функции (по требованиям).  
- **Макеты:** Figma (актуальные версии).  

- **Документация:** требования к веб‑ и мобильному приложениям.


## Используемые инструменты
- **Figma** — проверка соответствия макетам.  
- **Google Таблицы** — ведение чек‑листов, таблиц валидации, баг‑репортов.  
- **Браузер (Chrome/Firefox)** — тестирование веб‑версии.  
- **Эмулятор/устройство** — тестирование мобильного приложения.  


## Результаты
- **Веб‑приложение:**  
  - Протестировано 15 + сценариев (чек‑листы и валидация).  
  - Выявлено 8 багов (5 FAILED‑тестов, 3 дефекта вне документации).  
  - Все баги задокументированы с ID в баг‑репортах.  
- **Мобильное приложение:**  
  - Разработано 12 тест‑кейсов (включая проверку вёрстки).  
  - Обнаружено 2 дефекта вёрстки (зафиксированы в тест‑кейсах).
