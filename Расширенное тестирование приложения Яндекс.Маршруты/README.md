# Тестирование каршеринга в Яндекс Маршруты


## Цель
Проверить корректность работы функционала каршеринга (тариф «Роскошный»):  
- вёрстка ключевых блоков;  
- логика окон и кнопок;  
- соответствие требованиям.

## Выполнено
1. Подготовлены чек‑листы:  
   - на вёрстку (форма бронирования, карта с иконками);  
   - на логику окон («Способ оплаты», «Добавление карты»).  
2. Созданы тест‑кейсы для кнопки «Забронировать».  
3. Проведено тестирование по документации.  
4. Заведены баг‑репорты в Google Таблицу.


## Объект тестирования
- Форма бронирования.  
- Навигационная карта (иконки авто).  
- Окна «Способ оплаты» и «Добавление карты».  
- Кнопка «Забронировать».

## Инструменты
- Figma (макеты);  
- Google Таблицы (документация, баги);  
- тестовый стенд Яндекс Маршруты.

- Проверено **20+ сценариев**.  
- Выявлено **6 дефектов** (3 — вёрстка, 3 — логика).
